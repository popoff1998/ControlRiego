Ardomo Webserver Homepage

The following pages are available:

  • /index.htm - This page
  • /files.htm - Manage files on the server (browse or delete)
  • /$upload.htm - Built-in upload utility (upload individual files)

The following REST services are available:

  • /$sysinfo - Some system level information
  • /$list - Array of all files
  • /$update - OTA update firmware or filesystem
